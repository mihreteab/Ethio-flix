# moviedb-service

store and manage movie information on database

## Pre-installation

It's recommended that [NVM](https://github.com/creationix/nvm) be used to manage NodeJS versions.
The project includes an .nvmrc which specifies NodeJS 6.2.1

## Installation

```javascript
$ cd moviedb-service
$ nvm use
```

## Trial

```shell
$ npm start
```
